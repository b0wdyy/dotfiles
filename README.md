# Config files for NeoVim and Tmux

## Install a NerdFont
I use Hacker Nerd Font mono. You find more fonts [here](https://github.com/ryanoasis/nerd-fonts). 
You could also use a plugin which installs these fonts from the command line. [https://github.com/ronniedroid/getnf](https://github.com/ronniedroid/getnf)

## Install Tmux

1. `$ brew install tmux`
2. Open up a new session `tmux new`
3. Run `tmux source ~/.config/tmux/tmux.conf`
